## 主要功能

* 快速找到自己要使用的软件或文件。

##使用技巧

* 双击Alt键打开软件界面；
* 向搜索列表中添加软件或文件，将要添加的对象拖入软件界面即可；
* 当列出找到的软件或文件时，双击回车默认打开第一个对象。
